Our API's are in JSON format, and are free of cost under Open-Source Project


Geocoding Places

Ours is a very simple REST api, you only have to do a GET request to https://times-api-projjal/api/lat-long?city=CITY_NAME

Parameters

  • city:Name of the city

Sample Requests

The sample request for getting latitude and longitude information from our API for a given city:

http://times-api-projjal/api/lat-long?city=kolkata

Response

The result data is formatted using JSON. An example:

NOTE: Co-ordinates have been referenced from open.maprequest.api.com 

{
  "co-ord": {
    "lat": 22.567746, 
    "lng": 88.347602
  }, 
  "country-code": "IN", 
  "county": "West Bengal", 
  "map-url": "http://open.mapquestapi.com/staticmap/v5/map?key=peVM41X5wGi9vpUZfV1iOzzGcJGj1uFk&type=map&size=225,160&locations=22.5677459,88.3476023|marker-sm-50318A-1&scalebar=true&zoom=12&rand=-1422958136"
}

Get Sunrise Sunset Times

Ours is a very simple REST api, you only have to do a GET request to https://times-api-projjal/api/sunrise-sunset?city=CITY_NAME

Parameters

  • city:Name of the city

Sample Requests

The sample request for getting sunset and sunrise information from our API for a given city:

http://times-api-projjal/api/sunrise-sunset?city=kolkata

Response

The result data is formatted using JSON. An example:

NOTE: All times are in UTC and summer time adjustments are not included in the returned data.

{
  "astronomical_twilight_begin": "10:01:36 PM", 
  "astronomical_twilight_end": "2:21:06 PM", 
  "civil_twilight_begin": "11:02:23 PM", 
  "civil_twilight_end": "1:20:19 PM", 
  "day_length": "13:28:06", 
  "nautical_twilight_begin": "10:32:35 PM", 
  "nautical_twilight_end": "1:50:07 PM", 
  "solar_noon": "6:11:21 AM", 
  "sunrise": "11:27:18 PM", 
  "sunset": "12:55:24 PM"
}
